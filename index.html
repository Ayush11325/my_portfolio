Taking and enduring notes at the same time across all the devices is a tedious job today, but as we know Notes are the important part of our daily lives from business activities to some casual activity, we are managing notes for either for a reminder or some other formal purpose. NO8BOOK provides all these under one roof, always-connected system and comes with smart writing features which help in quick writing maintaining accuracy.
This project follows Agile MICRO-SERVICES software architecture which is achieved using RESTFUL API and developed using JAVASCRIPT FRAMEWORKS and PYTHON FRAMEWORKS. Word Generation achieved with the help of LSTM ( concept - based on SMART COMPOSE google research paper).
